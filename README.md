# Notas de resolución de algoritmos
## Problema de subarreglo que suma más
En una **lista** { $$x_1,x_2,x_3,...$$ } queremos tomar un sub arreglo tal que la suma de los valores sea la mayor posible.
Entonces una forma de resolver este problema es por *fuerza bruta*. haciendo un programa que consiga cada posible.
Otras formas de resolverlo eran:

- Divide y vencerás
- No me acuerdo el nombre de las otras maneras
- Gracias

Este fue el primer problema visto en clase
